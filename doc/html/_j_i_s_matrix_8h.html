inlinestatic

Get a row of indicated JIS char.

Parameters
utf8_s[in] JIS char (Char-Array) to get row.
out_buf[out] Pointer to store row data.
row_i[in] Row number between [0, 7].
Return values
size_tnumber of read char in utf8_s on success.
0on failure.
Note
*out_buf will not be changed when failure.
It caches most recent result. Serving for char same to previous is fast.
